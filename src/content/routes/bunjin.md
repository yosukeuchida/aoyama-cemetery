## このコースの楽しみ方

近代日本文学を代表する歌人・小説家・劇作家・思想家・評論家 12 名を、生年順(=活躍期の時系列順)で訪ねるコース。明治・大正・昭和の文学史を縦に辿れる。

- 1825-1900 / 明治の歌人 — 税所敦子(「明治の紫式部」、皇后の女官「楓内侍」、明治六歌仙)
- 1847-1901 / 自由民権思想 — 中江兆民(「東洋のルソー」、『三酔人経綸問答』)
- 1860-1945 / 国粋主義評論 — 三宅雪嶺(政教社を結成、雑誌『日本人』創刊で欧化主義に対峙した論壇の長老)
- 1868-1903 / 明治文壇の中心 — 尾崎紅葉(硯友社主宰、『金色夜叉』)
- 1871-1908 / 自然主義文学 — 国木田独歩(新聞記者として日清戦争に従軍)
- 1872-1939 / 新歌舞伎・捕物帳 — 岡本綺堂(『修禅寺物語』『半七捕物帳』、1937 年帝国芸術院会員)
- 1882-1953 / アララギ歌人 — 斎藤茂吉(精神科医、大久保利通と同じ 1種イ2号区画)
- 1883-1971 / 白樺派・小説の神様 — 志賀直哉(『暗夜行路』)
- 1888-1961 / 白樺派・劇作家 — 長與善郎(『項羽と劉邦』『青銅の基督』、長与専斎の子・長与又郎の弟)
- 1899-1951 / プロレタリア作家 — 宮本百合子(『伸子』『道標』、投獄を繰り返しながら創作)
- 1926-1997 / SF ショートショート — 星新一(1001 編を超える掌編)
- 1932-1999 / 戦後文芸評論 — 江藤淳(『夏目漱石』『成熟と喪失』)

約 110 分の散策で、12 名の文学者の墓を訪ねる順番がそのまま明治-平成の文学史の流れになる。志賀直哉・長與善郎・斎藤茂吉は大久保利通の周辺一帯に、星新一と江藤淳は近代政治家の区画から少し離れた静かな場所に、それぞれ独自の文学的世界を残して眠る。

注目してほしいのは、明治・大正期の対照的な 2 人の思想家。中江兆民は「東洋のルソー」と呼ばれた自由民権思想家、三宅雪嶺は政教社『日本人』誌で国粋主義を掲げた論壇の長老 — 自由民権と国粋主義、明治日本の思想を支えた二つの極を、近い区画で訪ねられる。宮本百合子は本コース唯一の女性小説家で、戦前の女性作家として最高峰の存在。投獄を重ねながら『伸子』『道標』を書き続けた経歴は、男性作家中心の近代文学史に女性の視点を加える。

長與善郎は衛生行政の祖・長与専斎の子で、東大総長 長与又郎の弟という近代日本のエリート一族から生まれた白樺派作家。志賀直哉と並べて訪ねると白樺派同志の関係が見えてくる。江藤淳は戦後文芸評論を代表する保守思想家で、星新一の墓と合わせて、近代から現代に至る文学評論・SF の系譜も辿れる。大久保利通(維新の政治家)と斎藤茂吉(歌人)が同じ 1種イ2号区画に眠るなど、政治史と文学史の重なりも肌で感じられるコース。
